Code Portfolio

Here you will find all of the projects that I have completed at my time at Ada.

Individual Projects:

Startrly

Startrly was a design product, where I created a webpage that matched a provided wireframe.

This introductory project into HTML and CSS allowed me to practice layout techniques (including CSS Grid and Flexbox)while working on producing DRY and readable CSS code by using various selectors.

Hotel

In Hotel, I built the codebase for a booking system for a small hotel.

Hotel was a cultimation project, in which I was able to use all that I have learned into a single project. The project required objected-oriented design and following TDD principles.

Following the completion of this project, I also had the opportunity to refactor my code following a code review.

Grocery Store

Grocery Store was a project designed to simulate a grocery store system where orders and customers are tracked.

In grocery store, I used Test-Driven-Development to write my code. In my project, I incorporated data from a CSV file, used instance variables and methods as well as object composition.

Solar System

A project that allowed the user to build a solar system model via the command line.

In this project, I worked on creating classes, corresponding methods, and code to allow the user to interact with the classes. With this code, I was working applying the Single Responsibility Principle to my classes.

Ride Share

A project where I created codebase that stored and used data on riders, drivers, and trips.

On this project, I worked creating nested data structures and understanding the logic to obtain the pieces of data in the nested structures while also working on iterating through the nested structures.

Portmanteau Generator

In Portmanteau Generator, I created a CLI that generated partmanteaus based on user input.

In writing this code, I used conditional logic and loops. I also worked on creating methods and variables and modifying variables.

Group Projects:

Slack CLI

Slack CLI was a project where we created a CLI that allowed the user to interact with the application Slack by posting messages and reviewing channels and users for a workspace.

Using TDD, we created a program that was able to send and receive responses from the Slack API. In our testing, we employed VCR to mock an external dependency.

This project was done in collaboration with Kelly Downes.

Object-Oriented Ride Share

Revisiting Ride Share, this implementation was object-oriented and included classes, methods and attributes.

Using TDD, this project reinforced creating classes with attributes, using composition to connect classes, and working with large amounts of data.

This project was done in collaboration with Tatiana Quintana.

Adagrams

Adagrams was a command line game where drawn letters are used to spell words. Each letter corresponded to a particular score and the highest final score was the winner.

The program required methods that were able to declare, read, and manipulate data. This was the first project I completed using pair programming.

This project was done in collaboration with Karla Guadron.

CS Fundamental Projects:

Factorial

Created a method to compute the factorial of a non-negative integer.

Integer Palindrome Check

Created a method to determine if a given integer was palindrome. Ruby provided functions where not used.

Palindrome Check

Created a method to determine if a given string is a palindrome without altering the original string. Only #length from Ruby provided String methods was used.

Reverse Sentence

Created a method where the words within a sentence were reversed. Only #length from Ruby provided String methods was used.

Reverse Words

Created a method where the letters in individual words within a sentence were reversed. Only #length from Ruby provided String methods was used.

String Reverse

Created a method where a string was reversed in place. Only #length from Ruby provided String methods was used.

Restricted Arrays

Various methods were created for a Restricted Array class where Ruby array functions were blocked. The methods created were #length, #print_array, #reverse, #search, #binary_search, #find_largest, #find_smallest.

Array Equals

Created a method that determined if two inputted integer arrays were equivalent. Only #length from Ruby provided Array methods was used.

Binary and Decimal

Created a method that converted an array of size 8 filled with 0s and 1s to the corresponding decimal value. The only used Ruby function was #length.
